# SecureDesk 🔐

SecureDesk es una aplicación web desarrollada en PHP para la gestión de incidencias de ciberseguridad.

El proyecto ha sido diseñado con fines académicos para un Trabajo de Fin de Grado (TFG), integrando análisis de seguridad estático (SAST) y dinámico (DAST).

---

## 🚀 Tecnologías

- PHP 8
- MySQL
- Bootstrap 5
- SonarCloud (SAST)
- OWASP ZAP (DAST)

---

## 📦 Instalación

1. Clonar repositorio:

```bash
git clone https://github.com/usuario/securedesk.git
```

2. Importar base de datos:

    database/database.sql

3. Configurar conexión:

    config/config.php

4. Ejecutar servidor:

    php -S localhost:8000
